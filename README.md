# Hackathon

## アプリ概要

- アプリ名：**COCO-ORU**

- 概要：金沢工業大学の自習室内の現在地情報共有アプリ

- ユーザー：自習室で友達と待ち合わせをしている人

- ニーズ：自習室内の現在地を待ち合わせている友達に送りたい

- ソリューション：いまいる席が写真として共有できるサービス
